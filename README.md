## 🚀 프로젝트 설치 및 실행

1. Git Clone

```plaintext
git clone https://github.com/developjik1/react-mobx-practice.git
```

2. 프로젝트 패키지 설치

```plaintext
npm install
```

3. 프로젝트 실행

```plaintext
npm start
```
